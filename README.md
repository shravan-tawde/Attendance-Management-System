# Attendance Management System

## Overview

Attendance Management System is a web-based application developed using Python, Flask, HTML, CSS, and JavaScript. It helps educational institutions manage student attendance efficiently through a simple and user-friendly interface.

## Features

* Secure Login System
* Student Attendance Tracking
* Multiple Class Management
* Attendance Reports and Analytics
* Search and Filter Functionality
* Data Export (TXT/CSV)
* Parent Portal Access
* Parent Notification System
* Holiday and Schedule Management
* Responsive User Interface
* Attendance Statistics Dashboard
* Real-Time Attendance Updates

## Installation

1. Clone the repository:
   git clone https://github.com/shravan132005/Attendance-Management-System.git

2. Navigate to the project directory:
   cd Attendance-Management-System

3. Install dependencies:
   pip install flask

4. Run the application:
   python app.py
